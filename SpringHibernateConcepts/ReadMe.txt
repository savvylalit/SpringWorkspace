Pre-requisites:
- MySql
- Java 1.8

Steps to run
1. There are 7 projects, demonstrating 7 concepts of Spring-Hibernate.

hb-01-one-to-one-uni
hb-02-one-to-one-bi
hb-03-one-to-many
hb-04-one-to-many-uni
hb-05-many-to-many
hb-eager-vs-lazy-demo
hibernate-tutorial

2. Each project contains a 'dbscript' folder, which consists of db scripts required for the project. Run those scripts before running the project.

3. Folder 'lib' in the root, consists of all the library files required to run all the project. Include all the jars from the lib into the classpath of each folder.
